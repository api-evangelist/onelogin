# OneLogin (onelogin)

OneLogin is an identity and access management platform providing single sign-on (SSO), multi-factor authentication, user provisioning, and a RESTful API for managing users, roles, applications, MFA, branding, connectors, reports, SAML assertions, smart hooks, and Vigilance AI.

## APIs

### OneLogin API

OneLogin REST API for identity and access management. The API follows RESTful principles, is secured by OAuth 2.0, and provides JSON messages with search, pagination, sorting, and filtering. Resources include Users, Roles, Apps, App Rules, Branding, Connectors, MFA, Reports, SAML Assertions, Smart Hooks, Smart MFA, User Mappings, Self- Registration, and Vigilance AI.
